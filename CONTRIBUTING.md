# Contributing to KUKA KRL Extension

First off, thank you for considering contributing to this project! It's people like you that make this tool better for the whole robotics community.

## How to Get Started

1. **Clone the repository**:
   ```bash
   git clone https://gitlab.com/LiskinLabs/kuka-krl-extension.git
   ```

2. **Install dependencies**:
   Navigate to the `extension` folder and run:
   ```bash
   cd extension
   npm install
   ```

3. **Running the extension**:
   - Open the project in VS Code.
   - Press `F5` to open a new window with the extension loaded.

## Code Standards

- **TypeScript**: We use strict typing. Avoid using `any`.
- **Linting**: Run `npm run lint` before committing. We use ESLint and Prettier.
- **Commit Messages**: Use descriptive commit messages (e.g., `feat: add inlay hints for signals`).

## Reporting Bugs

- Check if the bug is already reported in the [Issues](https://gitlab.com/LiskinLabs/kuka-krl-extension/-/issues) section.
- Provide a clear description of the problem and steps to reproduce it.
- Attach a sample `.src` or `.dat` file if possible.
